# Bitloop Calculator (Python)


A tool that calculates various data points given a bit string up to 28 bits.

The [front end][1] is accessible from the [Bitloops][2] page on my website, which goes into depth about what each data point means.

An HTTP GET request invokes an Amazon API Gateway and AWS Lambda function.

[calc.py][3] calculates the data returned for the first four sections.

* Properties of the bitstring ...
* Properties of the bitloop ...
* Properties of the tree ...
* Properties of the chain ...

[powerset.json][4] provides the data returned for the last section.

* Properties of the power set ...

This is generated by [the Java version][5].
